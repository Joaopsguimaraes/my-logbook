## 🧐 About <a name = "about"></a>

Para praticantes de musculação, entendem a importância das anotações de seus treinos, como qual treino vai fazer na segunda feira, quais exercícios, repetições e peso, com isso tenha um logbook para anotar suas cargas dos treinos anteriores para sempre progredir suas cargas ou acompanhar a evolução nos treinos.

## 🏁 Getting Started <a name = "getting_started"></a>

Para iniciar o projeto, instale os pacotes com o comando:

```shell
npm i
```

Logo após realize uma conexão com o banco de dados no arquivo .env, como tem exemplo o .env.example, e também informe as variáveis do clerk. Logo após, execute o comando:

```shell
npx prisma migrate dev
```

Para executar as migrations e seed.

Com isso a aplicação estará pronta para uso.

### Prerequisites

Node >= 22.11
